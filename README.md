### Бот парсер для провайдера astrainternet.ru


Бот проверяет трафик и в случае окончания нажимаем кнопку продлить, так-же проверяем баланс и отправляем его в телегу.

~~~~
Основными компонентами являются:

* needle
* cheerio
* jsdom
~~~~
